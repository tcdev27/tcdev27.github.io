Cleversteam: URI Gen

As part of my studies at university I have been tasked with a small JavaScript based system which allows the generation of a URI. The system is being developed to the specification set by Cleversteam, a web development company based in liverpool.

Part of the work Cleversteam does involves the creation of registration iframes that can be taken by customers and used withing different content management systems.

Numerous Problems arise with this as users often encounter errors caused by incorrect uris.

Enter the URI Gen! Which will take 3 input values and use them to produce a valid uri and then output it for the user to copy and paste it into their CMS.
