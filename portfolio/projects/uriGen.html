Cleversteam: URI Gen

The Project

Complete

As part of my studies at university I was tasked with creating a small JavaScript based system which allows the generation of a URI. The script is being developed to the specification set by a real web development company based in liverpool.

Part of the work the company does involves the creation of registration iframes that can be taken by customers and used withing different content management systems.

Numerous Problems arise with this as users often encounter errors caused by incorrect URI entries.

Enter the URI Gen! This form will take 3 key values from a user and then produce a URI for them to copy and paste into their CMS.

The Requirements:

The end product of this project needed to tick all of the following:

  • Single JavaScript file that can be dropped into any HTML5 page and run.
  • Adds a Form to the Page for the User to Fill in.
  • Generates a URI and Outputs it to the same page.
  • Allows the User to copy the URI to their clipboard to paste elswhere.
  • Validates the entered data and produces an output of Errors if any are found.

Which it does, and you can see for yourself with the Live Demo or Public Code Repository available via the links at the bottom of this page.

Methods & Technologies

My Development methodology for this project was a form of Waterfall. Where I identified the key requirements, coded to fulfill them and then tested the whole script as a Minimum Viable Product (MVP).

After the first set of tests, a small set of bugs were found as well as some extra features that could be added. This included more unique styling and addition of elements such as the 'Reset' button.

The bugs were fixed and the 'extras' were implemented to produce the final build.

The Generator was coded in vanilla JavaScript with the help of Gulp to produce a separate minified version that the client could deploy to improve performance.
